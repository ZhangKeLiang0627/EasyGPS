### 注意事项
本项目与原生项目的地图下载大相径庭，跟着说明走就行，只有几点需要注意一下：

- 地图瓦片格式转换时注意选择`Binary RGB565`，否则在EasyGPS上显示是反色的效果。

![](../../../4.Pics/map.png)

- SD卡的内存应当 ≤ 32GB，SD卡需要格式化为FAT32格式。
